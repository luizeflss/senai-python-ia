## 🚀 Senai: Python + Inteligência Artificial

Este repositório foi criado para centralizar, documentar e organizar todas as atividades práticas, relatórios de laboratório e desafios desenvolvidos durante o curso de **Python + Inteligência Artificial** no SENAI.

O objetivo principal deste espaço é registrar a evolução no aprendizado de lógica de programação, manipulação de dados com Python e, especialmente, o domínio de técnicas avançadas de **Engenharia de Prompts (Prompt Engineering)** utilizando modelos de linguagem de grande escala (LLMs).

---

## 📂 Conteúdo do Repositório

O repositório está estruturado para cobrir desde conceitos fundamentais até a automação de prompts complexos:

- **Laboratórios de Engenharia de Prompts:** Exercícios práticos testando variações de personas, restrições de tamanho (respostas curtas e longas), controle de idiomas, cruzamento de contextos complexos e detecção de limitações das IA.

- **Desafios Práticos com Frameworks:** Implementações em scripts Python automatizados utilizando frameworks estruturados do mercado para extrair o máximo de performance dos modelos de IA:

<div align="center">

| Framework | Significado |
|:---------:|:------------|
| ![RACE](https://img.shields.io/badge/RACE-c8102e?style=flat-square&logoColor=white) | *Role, Action, Context, Expectation* |
| ![CARE](https://img.shields.io/badge/CARE-a50000?style=flat-square&logoColor=white) | *Context, Action, Result, Example* |
| ![CREATE](https://img.shields.io/badge/CREATE-8b0000?style=flat-square&logoColor=white) | *Character, Request, Engage, Austere, Target, Evaluator* |
| ![CO-STAR](https://img.shields.io/badge/CO--STAR-700000?style=flat-square&logoColor=white) | *Context, Objective, Style, Tone, Audience, Response* |

</div>

- **Códigos em Python:** Integração com APIs de inteligência artificial (como o ecossistema OpenAI e endpoints da NVIDIA API) utilizando boas práticas de desenvolvimento, como o isolamento de credenciais com `python-dotenv`.
